Los proyectos más importantes en los que he participado

Consalud ENARM - 2017, 2019

Tecnologías: React, Express, Flutter, MySQL, PayPal

Es una plataforma LMS, destinada a los médicos egresados que buscan su residencia médica, la plataforma les permite prepararse para en examen ENARM

Para este proyecto dirigí y participé en el desarrollo de:

  • Aplicación en Android +10,000 descargas
  • Aplicación en IOS
  • Plataforma Web
  • Tienda Online
  • Dashboard
  • API

Paola Garza - 2020

Tecnologías: Angular, React, Laravel, MySQL, Stripe

Es una Single Page App, destinada a realizar valoraciones de la piel, fue desarrollada para la influencer Paola Garza e incluye un cuestionario y captura de fotografías para ser verificadas desde el dashboard por seres humanos para emitir un diagnóstico.

En este proyecto realicé todo el desarrollo, excepto el maquetado. El proyecto incluye:

  • Página de valoración (Solo para móviles)
  • Dashboard
  • API

SAUCCE MX 2015, 2017

Tecnologías: Angular, Laravel, MySQL, API Google Maps

Es uno de los proyectos más importantes en los que he participado, desarrollando completamente toda la estructura del ERP destinado a la gestión y verificación de documentos para los participantes en los apoyos del INADEM del 2015 al 2018.

Este proyecto fue desarrollado para el Centro de Capacitación de Ejecutivos y para Integra MiPymes Group, ambos colaboradores y proveedores del extinto Sistema Emprendedor. El proyecto incluye entre otras características:

  • Geolocalización de usuarios
  • Carga de documentos individuales
  • Carga de documentos masivos
  • Descarga por lotes
  • Generación de reportes
  • Generación de documentos en PDF
